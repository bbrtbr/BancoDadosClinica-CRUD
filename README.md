**Banco de Dados - Clínica Médica**

![clinica drawio](https://github.com/bbrtbr/BancoDadosClinica-CRUD/assets/109661421/f800f356-835c-4313-ba3c-8bcc8c3fd717)

''java
Requisitos: Diagrama Entidade Relacionamento 
Pelo menos 10 entidades
Especificação do banco de dados (detalhado, com justificativas para relacionamentos)
Sobre as justificativas, ver capítulos 2 e 3 aquelas seções que falam sobre escolhas. Ex.: atributo ou entidade? Atributo ou especialização? O que fazer com atributos opcionais? O que fazer com atributos multivalorados? Tem redundância?
Aplicar normalização de dados, ou utilizar as justificativas adequadas para não usar.
Pelo menos um relacionamento de cada tipo (aceitarei justificativas caso não seja possível ter de todos os tipos)
Pelo menos uma herança (generalização/especialização)
Tabelas com dados temporais
Modelo físico do banco de dados
Cada tabela deve ter pelo menos 10 registros
Devem haver pelo menos dois índices
Um sistema, em qualquer linguagem de programação, para manipular o banco de dados em modelo CRUD. O sistema pode ser tão simples quanto um programa em linha de comando. Assim, deve haver funções para
Criar dados (insert)
Ler dados (select)
Atualizar dados (update)
Apagar dados (delete)
Deve haver pelo menos uma função com join e uma com subconsulta aninhada.
'
